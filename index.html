DataDog Tools

DD-Sku-Tool

This tool helps you find and manage Datadog SKUs by selecting the appropriate quote type (Annual, Month-to-Month, On-Demand) and entering SKU descriptions. Easily copy the SKUs and their descriptions for further use.

DD ChatGPT Quote Terms Tool

This tool allows users to highlight additional services and service allotments, copy them into the chat, and receive them in a standardized format.

Infoblox Tools

IB-Order-Instruction-Tool

This tool simplifies creating Infoblox order instructions. Select the number of vendor quotes, input the required data, and copy the generated text for your email.

List Price Percentage Calculator

This tool simplifies the process of calculating the correct discount percentages for Carahsell. Simply enter the Infoblox SKU and the discount percentage off the list price, and the tool will automatically generate the Registered (Reg) and Non-Registered (NR) percentages required for Carahsell. It ensures accuracy and efficiency, allowing you to quickly copy and input the correct values into your quotes.
